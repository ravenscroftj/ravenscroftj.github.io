Project Summary

The TurboPilot project was an experimental runtime for getting open source, generative large-language-models to do code-completion on commodity computers without racks full of power-hungry GPUs. It was a learning opportunity for me to familiarise myself with Georgi Gerganov’s state-of-the-art llama.cpp quantized model runtime and served as an early proof-of-concept that local code completion is possible.

This project received a large amount of attention on twitter, reddit and hackernews and gained almost 4k stars on github. It was retweeted by Clem Delangue, CEO at Huggingface and featured in a keynote talk at Intel’s September 2023 Innovation Conference.

I decided to retire the project due to the ongoing improvement of llama.cpp which has made the need for a separate standalone runtime redundant. I wrote about my decision to retire this project here. I have left the source code for the project live in its original github project for posterity.
