Project Summary

Partridge was my first foray into text processing pipelines, starting as a final year undergraduate project back in 2012. The idea was to index scientific papers and make them easily to search by annotating each sentence with tags showing which core scientific concept1 the sentence corresponds to.

This makes it easy to carry out very targetted and specific literature searches. For example “Find biology papers where CRISPR is mentioned in a ‘Method’ sentence”.

Partridge hasn’t really been updated since I first built it in 2012-2013 and is in need of some TLC but it’s alive and kicking for anyone who wants to try it out.

  1. https://wiki.jamesravey.me/books/ml-and-data-science/page/core-scientific-concepts-coresc ↩︎
